## Advent of Code - Python

A Python template for Advent of Code.

## Getting Started

Refer to [flake.nix](./flake.nix) for the packages needed to compile this project.
If you have `nix`, you can run `nix develop` to enter a dev shell with all of the dependencies installed.

## Useful Commands

- Run tests:

  ```shell
  python -m unittest
  ```
- Run the program:

  ```shell
  python main.py
  ```

- Benchmark the program:
  ```shell
  hyperfine -N "python main.py"
  ```
